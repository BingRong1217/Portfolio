# 程式運行方式: 
* 需安裝 dosbox 運行，且因為運行環境為 masm，所以需要安裝 masm 有關文件，並且將 dos 掛載到 dosbox 的驅動器下。 
* 在使用 dosbox 之前，我們可以在 dosbox 的安裝文件中找到 dosbox 0.73 Options.bat 的末尾加上: 
mount c c:\dos 
path=%path%;\masm 
c: 
cd\asm 
* 打開 dosbox 時就會在 c 槽的 asm 資料夾，即可直接輸入遊戲名稱遊玩。 
# 遊戲介紹: 
* 進入遊戲開始介面後，輸入雙方的名字，按下 ENTER 即開始遊戲。 
* 遊戲開始時，兩人都各有五條命，利用 WASD 及方向箭上下左右來移動，及 G、K 來攻擊。 
* 遊戲中會隨機出現三種道具，愛心能增加一格生命；毒藥能在下一次攻擊對方時，讓對方扣 2 格生命；而盾牌則能在下一次遭到對方攻擊時不被扣生命。另外若碰到邊界的火，會一次扣光所有生命。 
* 將對方生命歸零即可獲得 1 分，若有一人到達3分則切換為下一個場景，遊戲總共有3個場景，3 個場景結束時會出現結束畫面，並顯示比分。
